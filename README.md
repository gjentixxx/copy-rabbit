# Copy & Rabbit

Copy & Rabbit is a lightweight web prototype for an automatic visual clipboard canvas. It turns copied text, links, images, videos, HTML, files, colors, prompts, code, and table data into draggable cards on an infinite canvas.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## Current prototype

- Infinite canvas pan and zoom
- Demo canvas with common copied content types
- Editable canvas names
- Copy, delete, and file download actions
- Live HTML and embed previews
- Drag-and-drop and paste capture support
